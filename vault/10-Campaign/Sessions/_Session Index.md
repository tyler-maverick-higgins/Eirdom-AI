# Session Index

## Summary

The running index of every session in *Platinum & Bramble* — both sessions already played (with links to their recap notes) and the next session in preparation. Each played session gets a numbered row pointing at its `type: session` recap; upcoming sessions are listed separately and marked as prep, not yet played, so a plan is never mistaken for a record of what happened at the table. Update this index whenever a session is prepped or recapped.

Sessions prepared but not yet played. These point at `gm-prep` notes, not session recaps. When played, move the entry down to *Played Sessions* with a link to its written recap.

## How To Use This Index

* **When prepping a session:** add a row to *Upcoming* pointing at the prep note.
* **After playing a session:** write the recap from [[Template - Session]], then move the entry from *Upcoming* into *Played Sessions* with the session number, date, title, and a link to the recap. Clear it from *Upcoming*.
* **Numbering:** session numbers are assigned in play order and do not change. The prep note for an upcoming session may carry its expected number, but it is not "Session N" in the played sense until it has been run.
